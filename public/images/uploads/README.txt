Drop your real screenshots here, then run:

npm run sync:screenshots

Required filenames:
- asaa-desktop.png (or .jpg/.jpeg/.webp/.svg)
- asaa-mobile.png
- myfinance-desktop.png
- myfinance-mobile.png
- barakatrack-desktop.png
- barakatrack-mobile.png

The script will:
1) Copy images to public/images with portfolio naming.
2) Update src/data/projects.ts automatically.
